[대전산단] 2026 AX 실증산단 구축사업 컨소시엄 안

노후산단 영내거래 활성화의 ‘OASIS’,
우주·국방 신뢰사슬 ‘AXIS-CHAIN’ 구축

대전산업단지의 입주기업을 대상으로 온톨로지 기반 적응적 공급망 매칭(OASIS)과 항상-동작 센싱 사슬(AXIS-CHAIN)을 결합한 『대전 AX 실증산단』 구축

사업 개요

  • (규모) 총 196억원(국비 140억, 매칭 56억 / 매칭 = 지방비 + 민간부담금 + 산단공 자산 현물), ‘26 ~ ‘29년 (4년간)
    • 매칭은 정부출연금 대비 40% 이상 의무 (공고 기준)
    • 사업기간 종료 후 성과활용 5년 이상 운영 의무
  • (지원대상) 대전산업단지(거점산단) + 대덕연구개발특구·연계산단
    • 대전산단 입주기업 약 970개, 생산액 4조원 / 대덕특구 약 2,347개, 생산액 19조원
    • 대전산단은 50년된 노후산단으로 기계·금속 기반 전통제조 58%, 영내거래·디지털 전환 시급
  • (사업주체) (주관) 한국전자기술연구원(KETI)
    • (참여 / 앵커기업) ㈜링크솔루션 · ㈜액스비스
    • (참여 / 기관) 대전TP · 한밭대 · 한국산업단지공단 충청권본부 (필수참여, 본 공고 자격요건)
    • (협력) 산업통상부 M.AX 얼라이언스 산단AX분과 · 대전광역시

주요 내용

  • OASIS(Ontology-Adaptive Supply-chain Intelligent System) — 노후산단 영내거래 활성화의 오아시스
    • 온톨로지(T-Box·A-Box) + Multi-agent LLM 라우팅으로 수요-공급 자동 배정
    • ‘영내 거래 활성화 에이전트’ 별도 탑재 — 공공성 시스템적 답변 메커니즘
    • 다크팩토리(㈜링크솔루션) 코어 + 후가공·열처리·후처리 기업 풀 연계
  • AXIS-CHAIN(Always-on eXchange Intelligence Sensing Chain) — 우주·국방 신뢰사슬
    • 5종 센서 노드(플라즈마광·음향방출·진동·비전·열화상) + 엣지 AI + Mesh 협동
    • 노후 장비를 교체하지 않고 AI 모듈만 부착 → 산단 기업 즉시 편입
    • ㈜액스비스 고정밀 장비 노하우 + 시계열 파형 분석 강점 활용
  • 두 축이 만들어내는 단일 신뢰 사슬 — 우주·국방 부품 인증의 핵심 요건인 End-to-End Provenance(전 과정 이력추적) 자동 생성
  • 5극3특 정합 : (중부권 코어) 대전산단 → (충청권 확장) 충북·충남 → (전국 확장) 5극3특 + 강원후평 MINI 자동 정합

추진 전략 핵심 5축

  1. (공공성 시스템화) OASIS의 4번째 에이전트 = 영내거래 활성화 에이전트 → 산단공 핵심 지적(“공급은 있는데 수요가 없다”)에 시스템적 답변
  2. (우주·국방 fit-for-purpose) 두 축이 자동 생성하는 단일 신뢰사슬 = 우주·국방 부품 인증의 4중 요건 자동 충족
  3. (5극3특 권역 확장 메커니즘) 온톨로지 A-Box 인스턴스 추가만으로 신규 기업·산단 즉시 편입 → 시스템 자체가 권역 확장 메커니즘
  4. (M.AX 얼라이언스 자연 결합) 본 컨소시엄 선정 = 대전 MINI 얼라이언스 자동 출범, 1차 10개 MINI(특히 천안·강원후평)와 권역 협력 자동 발생
  5. (자립화 BM 다중 구조) 8종 수익원 + 5년 누적 124억 → 권역 확장이 매출 증가로 직결되는 자연 BM
